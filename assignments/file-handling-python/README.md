# 📘 Assignment: File Handling and Error Handling in Python

## 🎯 Objective

Practice reading from and writing to files in Python while using simple error handling to make programs more reliable.

## 📝 Tasks

### 🛠️ Read and Display a File

#### Description
Write a small program that opens a text file and prints each line to the screen.

#### Requirements
Completed program should:

- Open a file using Python
- Read the contents line by line
- Print the results clearly to the user

### 🛠️ Save User Input to a File

#### Description
Create a program that asks the user for a short note and saves it into a text file.

#### Requirements
Completed program should:

- Prompt the user for input
- Write the input to a file named `notes.txt`
- Handle missing files or write errors safely
