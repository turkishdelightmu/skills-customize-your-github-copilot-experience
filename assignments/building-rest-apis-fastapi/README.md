# 📘 Assignment: Building REST APIs with FastAPI

## 🎯 Objective

Build a small REST API in Python using FastAPI. Students will practice creating routes, handling request data, and returning JSON responses in a simple web application.

## 📝 Tasks

### 🛠️ Create the FastAPI App

#### Description
Set up a basic FastAPI application with a home endpoint and a simple collection of sample items.

#### Requirements
Completed program should:

- Import FastAPI and create an app instance
- Add a root endpoint that returns a welcome message
- Add an endpoint that returns a list of sample items in JSON format

### 🛠️ Add CRUD-style Endpoints

#### Description
Extend the app with endpoints for reading and creating items.

#### Requirements
Completed program should:

- Add an endpoint to return one item by ID
- Add an endpoint to create a new item
- Return clear JSON responses for both success and input cases
